Flight Search API + others (Lets Travel App)

In this project we called up to 5 API to determinated Weather, track User's IP, cheap flights and images of destinations. The focus of this application was in it's backend.

I selected the origin of our travelers by identifying their public IP and the destination by generating an array of multiple states, after the client clicks on the button (id = btn-ClientSubmit) it generates a random element from the "destination" array, given us our target destination (note: this might not be the only random element selected, this function can be triggered multiple times if the api respond with "no tickets found for the destination" in which in that case our recursion funtion will trigger the whole function (flightserach) again if an error is found)
